# Personal-Diary
A simple command-line Personal Diary Application written in Python.
This project allows users to create, read, update, and delete diary entries stored as text files.
It is beginner-friendly and a great project to practice:
File handling
Input validation
Folders & paths (os module)
Loops & menu-driven programs
Exception handling
Python’s match statement

✨ Features
✅ Create a new diary entry
Save entries by date (ddmmyyyy format)
Automatically creates a dedicated diary/ folder
Prevents overwriting if the file already exists

✅ Read an existing entry
Shows a list of saved entries
Displays the full content of the selected diary file

✅ Add to an existing entry
Append new thoughts or updates to a previous entry

✅ Delete an entry
Shows available files
Deletes the selected diary file safely

✅ Exit the program
